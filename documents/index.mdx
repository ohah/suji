# Suji

Zig 코어 기반의 올인원 데스크톱 앱 프레임워크. Electron의 IPC 모델을 Zig/Rust/Go/Node.js
4개 백엔드로 확장.

## 빠른 시작

```bash
npx @suji/cli init my-app --backend=multi
cd my-app
suji dev
```

## 설계 원칙

- **Electron 호환 계층**: 친숙한 네이밍 (`invoke`, `on`, `send`, `handle`)
- **크로스-언어**: 같은 IPC 포맷을 Zig/Rust/Go/Node 백엔드가 공유
- **허브-스포크**: 렌더러는 코어(메인) 프로세스만 거쳐 통신 (직접 renderer↔renderer X)
- **JSON 와이어**: 모든 IPC payload는 JSON — 언어별 직렬화 차이 없음

## 지원 플랫폼

- macOS (Apple Silicon + Intel)
- Linux (x86_64)
- Windows (x86_64)
